# desafio-super-trunfo
"Primeiro desafio do curso Super Trunfo de Países em C — cadastro de cartas com entrada e saída de dados."

Este é o primeiro projeto do desafio **Super Trunfo de Países**, desenvolvido em linguagem C.  
O objetivo é praticar os fundamentos de entrada, processamento e saída de dados, criando um sistema simples para cadastrar **duas cartas** com informações sobre cidades (estado, código, população, área, PIB, pontos turísticos etc.).

## Funcionalidades
- Cadastro sequencial de duas cartas  
- Leitura de dados via teclado (`scanf`)  
- Exibição organizada das informações cadastradas (`printf`)
